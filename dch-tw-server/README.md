## Description

## Setup Requirements

- MongoDB, at least v3.6.8
- Node, at least v10.18.1
- Npm, at least v6.13.4
- Twitter DEV access tokens


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
